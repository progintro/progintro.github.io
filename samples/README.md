# Παραδείγματα README και λύσεις ασκήσεων

Τα παρακάτω παραδείγματα αποτελούν πραγματικές υποβολές από φοιτητές / φοιτήτριες και μπορούν να σας δώσουν μια ιδέα για το πως φαίνεται ένα τελικό README μαζί με τον κώδικα που το συνοδεύει. Δεν είναι όλες οι λύσεις αυτές τέλειες (σε κάποιες ο κώδικας είναι καλύτερος, σε άλλες το README), αλλά η συνολική τους ποιότητα ήταν τέτοια ώστε να θεωρηθούν υποδειγματικές. Η λίστα δεν είναι πλήρης - δυστυχώς δεν φτάνει ο χώρος (και ο χρόνος!) για να συμπεριλάβουμε όλες τις εξαιρετικές υποβολές που δεχόμαστε.
