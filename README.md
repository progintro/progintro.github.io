* Τμήμα #3 - Τρίτη 13:00-15:00 (Linux)
* Τμήμα #4 - Τρίτη 15:00-17:00 (Windows PC1)
* Τμήμα #5 - Τετάρτη 15:00-17:00 (Windows PC1)
* Τμήμα #6 - Τετάρτη 17:00-19:00 (Windows PC1)
